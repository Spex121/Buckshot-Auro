# 🔫 Auro Script
### Terminal assistant for Buckshot Roulette

[English](#english) · [Русский](#русский)

## English

**Auro Script** is a lightweight terminal tool for **Buckshot Roulette**.  
It tracks shells in the chamber and displays real-time shot probability after every move.

### ⚙️ Features
- 🔴 Track live and blank shells
- 📊 Real-time shot probability in %
- 🌍 English and Russian interface
- 🛡️ Handles all input errors gracefully

### ✅ Advantages
- **Simple** — launch and play instantly
- **Lightweight** — zero dependencies
- **Terminal** — runs anywhere

### 📖 Usage
1. Select language
2. Enter shells → `Combat/Blank` (e.g. `4/2`)
3. Press `+` for combat shot, `-` for blank
4. Watch probability update in real time

## Русский

**Auro Script** — лёгкий терминальный инструмент для **Buckshot Roulette**.  
Отслеживает патроны в барабане и показывает шансы в реальном времени после каждого выстрела.

### ⚙️ Умеет
- 🔴 Считать боевые и холостые патроны
- 📊 Показывать шанс следующего выстрела в %
- 🌍 Работать на русском и английском языке
- 🛡️ Корректно обрабатывать любые ошибки ввода

### ✅ Преимущества
- **Простота** — запустил и играешь
- **Лёгковесность** — никаких зависимостей
- **Терминальный** — работает везде

### 📖 Использование
1. Выбери язык
2. Введи патроны → `Боевые/Холостые` (например `4/2`)
3. Нажми `+` боевой, `-` холостой
4. Смотри шансы в реальном времени
